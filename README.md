# RadioControlSoftware
## Output Package
| Channel | Value |
| --- | --- |
| 0 | Right-X |
| 1 | Right-Y |
| 2 | Left-X |
| 3 | Left-Y |
| 4 | Flightmode |
| 5 | Armed |
| 6 | Empty |
| 7 | Empty |

All Joystick values are between 0 and 255 with 127 beeing center.
